# Image Upload Instructions

## Required Images

Upload these images to your Supabase `listing_images` bucket:

### Main Images
- `hero-nature-cabin.jpg` - Hero background image
- `cabin-type.jpg` - Cabins category image
- `treehouse-type.jpg` - Treehouses category image
- `glamping-type.jpg` - Glamping category image
- `tinyhouse-type.jpg` - Tiny Houses category image
- `farm-type.jpg` - Farm Stays category image

### Listing Images
- `cabin-1.jpg` through `cabin-5.jpg` - Cabin listing images
- `treehouse-1.jpg` through `treehouse-5.jpg` - Treehouse listing images
- `glamping-1.jpg` through `glamping-5.jpg` - Glamping listing images
- `farm-1.jpg` through `farm-5.jpg` - Farm listing images

## Upload Instructions

1. Go to your Supabase dashboard
2. Navigate to Storage > Buckets > listing_images
3. Click "Upload" and select all the images from the temp_images folder
4. Ensure all images have public access policies
